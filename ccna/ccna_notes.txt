I. Chapter 1
  A. TCP/IP and OSI Model: Data Encapsulation (Page 6) 
    1. Application: End-User Input "DATA"
    2. Transport (Segment): Data encapsulated with TCP, (TCP) encapsulated with TCP Header 
    3. Network (Router): Connect Networks, (TCP + DATA) encapsulated with "IP Header" Packets = (IP + TCP + DATA)
    4. Data Link (Switch): Connect End-Users, (IP + TCP + DATA) encapsulated with Frames = (IP + TCP + DATA + FRAME), 
       Frames contain Mac Addresses
      a. Retains the last 10 commands
      b. Frames contain "MAC Addresses"
      c. Purpose is to learn Mac Addresses
      d. 
    5. Physical (Hub): Connect End-Users, Problem "Collisions", Sent as Bits
      a. 
  B. Packet Tracer
    1. Configuration
      a. RED connection on PT = Port Shut by Default
    2. TCP/IP
    3. Switch Commands 
      a. "enable" = Privilege Mode
      b. "conf t" = Configure Terminal
      c. "int fa0/10" = Configure Interface
      d. "?" = list command line options
      e. "clear mac-address-table" = clear table
      f. "sh mac-a" = show

## Subnetting with Tool #1 and Tool #2 MN

192.168.4.70 /26 mn = 64
255.255.255.192 [STEP 1] = Mask
192.168.4.64 [STEP 2] = Network ID "64" derived by 70/64 = approx. 1 and 64*1=64
192.168.4.(64 + 1) = 192.168.4.65 = First Valid +0.0.0.1
192.168.4.(127 - 1) = 192.168.4.126 = Last Valid -0.0.0.1 
192.168.4.(128 - 1) = 192.168.4.127 = Broadcast  IP -0.0.0.1
192.168.4.(64 + mn) = 192.168.4.128
/32 - /26 = 6 
2^6 = 64 - 2 = 62 Hosts 

192.168.1.42 / 29   mn = 8
255.255.255.248
192.168.1.40 and "40" = 42 / 8 = 5 and 5*8
192.168.1.(40 + 1) = 192.168.41 = First Valid
192.168.1.(47 - 1) = 192.168.1.46 = Last Valid
192.168.1.(48 - 1) = 192.168.1.47 = Broadcast IP 
192.168.1.(40 + mn) = 192.168.1.48
/32 - /29 = 3 
2^3 = 8 - 2 = 6 Hosts


192.168.23.61 /28   mn = 16
255.255.255.240
192.168.23.48  (61 / 16 = 3 and 3*16 = 48)
192.168.23.49 First 
192.168.23.62 Last
192.168.23.63 Broadcast
192.168.23.64 (48 + 16) Next Address/Subnet

172.16.159.159 /22  mn = 4
255.255.252.0 
172.16.156.0 (159 / 4 = 39*4) = 156
172.16.156.0 + 0.0.0.1 = 172.16.156.1 First Valid
172.16.159.255 - 0.0.0.1 = 172.16.159.254 Last Valid
172.16.160.0 - 0.0.0.1 = 172.16.159.255 Broadcast
172.16.(156 + 4).0 = 172.16.160.0
/32 - /22 = 10
2^10 = 1024 - 2 = 1022 Hosts

172.19.20.23 /28  mn = 16
255.255..255.240
172.19.20.16 (23 / 16 = 1*16) = 16
172.19.20.17 First Valid
172.19.20.30 Last Valid
172.19.20.31 Broadcast 
172.19.20.(16 + 16) = 172.16.20.32 Next Address
/32 - /28 = 4
